BASE DE DATOS
1. Ejecuta el archivo database.sql o bien copia y pega su contenido en el gestor MySql que tu tengas

BACKEND
1. Ingresa al directorio desde terminal con el comando cd Backend y ejecuta el comando npm i
2. Crea un archivo .env con los datos que estan en el archivo .env.example
3. Corre el servidor con node con el comando npm run dev

FRONTEND
1. Ingresa al directorio desde otra nueva terminal con el comando cd Frontend y ejecuta el comando npm i
2. Ejecuta el proyecto con el comando npm run dev
